# Assets Directory

This directory contains icons and other assets for the Deadlock Prevention & Recovery Toolkit.

## Structure

- `icons/` - SVG and PNG icons for processes, resources, and UI elements
- `images/` - Screenshots and example visualizations
- `data/` - JSON files for predefined scenarios and examples

## Usage

Icons can be referenced in the application using relative paths:
```html
<img src="assets/icons/process-icon.svg" alt="Process">
```
